## ğŸ“– Overview
This project explores the funding ecosystem of Indian Startups between **2018 and 2025**. 
It aims to provide a comprehensive analysis using Data Cleaning, Exploratory Data Analysis (EDA), Visualization, and Machine Learning to uncover trends in the Indian market.

**Key Goals:**
* Identify which **industries** and **cities** are receiving the most funding.
* Analyze **year-wise investment trends** (Funding Winter vs. Boom).
* Track the growth of **Top Investors**.
* Build a **Predictive Model (Linear Regression)** to estimate future funding amounts.
